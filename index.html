Tentang Repository

Repository ini berisi berbagai latihan yang dilakukan selama proses pembelajaran Git dan GitHub, termasuk:

  • Cara menginisialisasi repository Git lokal
  • Cara melakukan perubahan, commit, dan push
  • Cara membuat branch baru dan menggabungkannya (merge)
  • Cara berkolaborasi melalui pull request
  • Menggunakan fitur GitHub lainnya untuk kolaborasi

Tujuan Pembelajaran

  • Memahami dasar-dasar penggunaan Git (init, commit, push, pull).
  • Memahami cara menggunakan GitHub untuk mengelola repository.
  • Memahami konsep branching dan merging.
  • Menguasai cara upload tugas ke GitHub untuk penyimpanan dan kolaborasi.

Struktur Folder

  • /latihan: Berisi berbagai latihan dasar Git.
  • /tugas: Folder khusus untuk menyimpan tugas yang di-upload ke GitHub.
  • README.md: File ini berisi panduan dan informasi tentang repository.

Flow Metaflow Informatika UMS

Penjelasan Tiap Langkah

  • start: Memulai alur akademik dengan pengenalan awal.
  • pmb: Tahap Pengenalan Mahasiswa Baru (PMB).
  • krs_planning: Tahap pengisian Kartu Rencana Studi (KRS) untuk memulai semester.
  • semester_journey: Mengikuti alur perkuliahan di setiap semester, termasuk perkuliahan, UTS, proyek/praktikum, dan UAS. Mahasiswa diminta untuk menginput nilai IP setiap semester.
  • skripsi_ta: Tahap Skripsi atau Tugas Akhir (TA). Jika mahasiswa tidak lulus, akan mengulangi proses ini.
  • graduation: Menghitung IPK (rata-rata nilai IP dari semua semester) dan persiapan untuk wisuda.
  • end: Menyelesaikan alur perkuliahan.

Menjalankan Flow

Untuk menjalankan flow, simpan kode Python ini di file bernama Informatic_Student_College_UMS.py. Jalankan dengan perintah berikut di terminal:

python Informatic_Student_College_UMS.py

Cara Menggunakan

Untuk memulai, lakukan clone repository ini ke komputer lokal Anda:

git clone https://github.com/ums-l200220279/ums-l200220279.github.io.git

Setelah itu, buka folder yang berisi file Informatic_Student_College_UMS.py dan jalankan kode tersebut sesuai dengan instruksi di atas.

Kontribusi

Jika Anda ingin berkontribusi pada proyek ini, Anda dapat melakukan fork repository ini, membuat perubahan di branch Anda, dan kemudian mengajukan pull request. Pastikan untuk mengikuti aturan dan konvensi yang ada.
